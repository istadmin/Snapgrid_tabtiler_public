## Summary (132 chars max)

## Description

Stop wasting time dragging and resizing browser windows manually. SnapGrid is a lightweight, privacy-first window manager that organizes your tabs and windows into perfectly aligned grid layouts with a single click.

Whether you are a developer comparing code side by side, a researcher cross-referencing sources, or a student balancing lecture notes with assignments, SnapGrid helps you work smarter across your entire screen.

LAYOUTS

Choose from five flexible grid arrangements:

- 1x2 — Two windows side by side (horizontal split)
- 2x1 — Two windows stacked vertically
- 2x2 — Four windows in a quad grid
- 3 Rows — Three windows stacked in horizontal strips
- 3 Columns — Three windows in vertical columns

TAB PICKER

Select exactly which tabs to arrange from a visual list grouped by browser window. Each tab shows its favicon and title for quick identification. The picker automatically limits your selection to match the chosen layout.

DRAG AND DROP

After arranging windows, use the built-in position grid to drag and drop windows between slots. Swap two windows instantly or move a window to an empty slot — no resizing needed.

TILE EXISTING WINDOWS

Already working across multiple browser windows? Tile them into your chosen grid layout without splitting or moving tabs. SnapGrid repositions and resizes your existing windows to fill the screen.

MERGE BACK

Done multitasking? Merge all open windows back into a single maximized browser window with one click.

MULTI-MONITOR SUPPORT

SnapGrid automatically detects which monitor your active window is on and tiles within that display's available work area. It respects your taskbar position and works seamlessly across any number of connected monitors.

RIGHT-CLICK CONTEXT MENU

Access all five layout modes and the merge option directly from the extension icon's right-click context menu for faster workflow.

PRIVACY BY DESIGN

SnapGrid collects zero user data. No analytics, no tracking, no telemetry, no external server communication. Your browsing activity stays entirely private. See our privacy policy for full details.

BROWSER COMPATIBILITY

Built with Manifest V3 and fully compatible with:
- Google Chrome (version 88 and above)
- Microsoft Edge (version 88 and above)
- Brave, Opera, Vivaldi, and other Chromium-based browsers

IDEAL FOR

- Software developers — compare code, debug across editor and browser, monitor logs
- Students and researchers — lecture notes, PDFs, reference material side by side
- Content creators — manage social media dashboards, image editors, writing tools
- Finance professionals — monitor multiple dashboards, charts, and reports
- Remote workers — video calls alongside documents and collaboration tools
- Power users — anyone who multitasks with browser tabs daily

HOW TO USE

1. Click the SnapGrid icon in your browser toolbar
2. Choose a layout: 1x2, 2x1, 2x2, 3 Rows, or 3 Columns
3. Select the tabs you want to arrange using the tab picker
4. Click "Arrange Selected Tabs" to snap them into place
5. Drag and drop windows in the position grid to rearrange
6. Click "Merge All to One Window" when you are done
